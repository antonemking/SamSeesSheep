Methodology
· Model: ultralytics YOLOE-seg, text prompts "sheep head" / "sheep ear" / "sheep nose" — no fine-tuning
· Tracking: Ultralytics built-in (ByteTrack) — stable per-sheep track IDs across frames
· Ear angle: nose → ear-midpoint for the dorsal axis (PCA fallback); ear vector from base → tip
· Thresholds: McLennan & Mahmoud 2019 (SPFES), Reefmann 2009, Boissy 2011
· This repo exists to measure out-of-the-box YOLO performance on sheep anatomy. Custom training lives in a separate repo.
